Installation Guide

Follow these steps to load the extension manually

1

Download & Unzip

Click the download button above to get the ZIP file. Extract it to a folder you'll keep — the extension loads from this folder, so don't delete it after installing.

2

Open Extensions Page

In your browser, navigate to the extensions page:

  • Chrome: chrome://extensions
  • Edge: edge://extensions
  • Brave: brave://extensions
3

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the extensions page. This allows you to load unpacked extensions.

4

Load the Extension

Click "Load unpacked" in the top-left corner, then select the folder you extracted in step 1. Duly Noted will appear in your extensions list.

5

Pin & Launch

Click the puzzle piece icon in your toolbar and pin Duly Noted for quick access. Press Alt+Shift+V to open the side panel instantly from any tab.

Common Questions

Will I get updates?

Once Duly Noted is approved on the Chrome Web Store, you can switch to the store version for automatic updates. Until then, re-download and re-load to update.

Is this safe?

Absolutely. Duly Noted is open source — you can review the code on GitHub. All transcription happens locally on your device. No data is sent to external servers.

Why "Developer mode"?

Chrome requires Developer mode to install extensions outside the Web Store. This is standard practice for beta testing and is completely safe.

What if I see a warning?

Chrome may show a "Disable developer mode extensions" popup on startup. Just click the dismiss option. This goes away once the extension is on the Chrome Web Store.
